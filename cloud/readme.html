TIBCO BusinessEvents Cloud Tools

To deploy and monitor your TIBCO BusinessEvents applications on different cloud platforms by using Docker and Kubernetes, you can use tools provided by TIBCO on the GitHub repository at https://github.com/TIBCOSoftware/be-tools/tree/master/cloud.

To use these tools and scripts for your TIBCO BusinessEvents applications, clone or download the "cloud" folder from the GitHub repository to BE_HOME. For more information on how to use those scripts after copying them to BE_HOME, see TIBCO BusinessEvents Cloud Deployment Guide.
