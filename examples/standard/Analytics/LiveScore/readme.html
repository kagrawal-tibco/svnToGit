LiveScore Example

Purpose of This Example

This example shows you how to use the TIBCO BusinessEvents® LiveScore Example. SOAP request is sent to LiveScore according to the properties given in the Event

This request calls a LiveScore API and returns a response which is the result of the Test data run on the model in LiveScore workspace created given by WFID.

Instructions

Before you begin, ensure that the EAR file you want to use is in the BE_HOME/examples/standard/LiveScore folder, where this readme is also stored.

  1. Open a command window and at the command prompt, start the TIBCO BusinessEvents engine using the following command. Substitute your actual value for BE_HOME.

    cd BE_HOME/examples/standard/Analytics/LiveScore
BE_HOME/bin/be-engine --propFile BE_HOME\bin\be-engine.tra -u default -c LiveScore/LiveScore.cdd LiveScore.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  2. Send a Event to invoke LiveScore API

    In the command window you see a message like this: Sending Statistica Event

    Answer received from the event is: { ... }

  3. To reset the data, restart the engine at the command line.
