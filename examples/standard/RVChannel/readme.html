Rendezvous Channel

Purpose of This Example

The RVChannel example demonstrates how TIBCO BusinessEvents® uses a Rendezvous® channel to send and receive events using TIBCO Rendezvous®. You can examine the project in TIBCO BusinessEvents Studio to understand the setup, and refer to TIBCO BusinessEvents Developer's Guide for details on channel configuration.

Instructions

Configure the Example

Before running the example, do the following setup.

  1. Edit one environment variable in BE_HOME/bin/be-engine.tra to point to your Rendezvous installation: 

    tibco.env.RV_HOME=yourRV_HOME

  2. Similarly, edit Compile.bat in the RVSend directory to point to your Rendezvous and Java installations: 

    set RVHOME=yourRV_HOME
    set JRE_HOME=yourJRE_HOME

  3. At a command prompt, run Compile.bat to compile the RVSend Java class.

    cd BE_HOME/examples/RVChannel/RVSend
Compile.bat

Run the Example

Before you begin, ensure that the example EAR file is in the BE_HOME/examples/standard/RVChannel folder, where this readme is also stored.

  1. Open a command window and at the command prompt, start the TIBCO BusinessEvents engine using the following command.

    cd BE_HOME/examples/standard/RVChannel
BE_HOME/bin/be-engine --propFile BE_HOME/bin/be-engine.tra -u default -c RVChannel/RV.cdd RVChannel.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  2. Make sure the command window showing the TIBCO BusinessEvents engine is visible, and that the text at the bottom is similar to BE Engine host-user started. Then in the other command window, run the Java class, as shown below, to send the Rendezvous message. You can substitute your own string values for BookTitle, BookAuthor and BookPrice, as desired:

    cd BE_HOME/examples/standard/RVChannel/RVSend
java tibrvmultisend Subject.BookData BookTitle="Succeed in Life" BookAuthor="Ima Success" BookPrice=300

  3. Send more messages as desired, using different details. For example:

    java tibrvmultisend Subject.BookData BookTitle="Harry Potter" BookAuthor="Rowling" BookPrice=150

  4. A message arriving at the BookDataDestination destination within the Rendezvous channel (RVChannel) is transformed into a BookDataEvent which is asserted. The BookInformationRule executes on assertion of this event. It prints the details of the message to the command window. For example:

    #########Publishing Book Information##########
Book 'Succeed in Life' is written by Ima Success.
Its market price is just $300
