OneExternalEventAndTwoConcepts

Purpose of This Example

This TIBCO BusinessEvents® example demonstrates an external event working on two different concepts. A rule matches the checking account and credit account from the working memory to the ones given on the payment event. The rule then applies the debit to checking account and credit to the credit account, simulating a credit card bill payment.

How to Run the Example

  1. Open a command window and at the command prompt, start the TIBCO BusinessEvents engine using the following command.

    cd BE_HOME/examples/standard/OneExternalEventAndTwoConcepts
BE_HOME/bin/be-engine --propFile BE_HOME/bin/be-engine.tra -u default -c OneExternalEventAndTwoConceptsRepo/OneExternalEventAndTwoConceptsRepo.cdd OneExternalEventAndTwoConceptsRepo.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  2. Create a Bill Payment Event and give the values as "Checking1" and "Credit1" for the checking and credit accounts respectively.
