MultipleSessionsAndLocalChannel

Purpose of This Example

This example demonstrates use of multiple agents (also known as sessions) and a local channel in TIBCO BusinessEvents®.

This example shows how you do the following:
  1. Create multiple TIBCO BusinessEvents agents using the CDD file.
  2. Create a local channel.
  3. Route to an agent through the local channel.

How to Run the Example

  1. Open a command window and at the command prompt, start the TIBCO BusinessEvents engine using the following command. Substitute your actual value for BE_HOME.

    cd BE_HOME/examples/standard/MultipleSessionsAndLocalChannel
BE_HOME/bin/be-engine --propFile BE_HOME/bin/be-engine.tra -u default -c MultipleSessionsAndLocalChannelProject/MultipleSessionsAndLocalChannelProject.cdd MultipleSessionsAndLocalChannelProject.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  2. Create an Order Event and give it a value as desired.
