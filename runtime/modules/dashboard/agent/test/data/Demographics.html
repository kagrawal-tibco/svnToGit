Demogrphics Dashboard

Note   The TIBCO BusinessEvents® Express edition does not support use of the TIBCO BusinessEvents® Views add-on.

Purpose of This Example

This example shows a simple dashboard which show tickers with thier price and volume

Prerequisites

  • A JDBC backing store configured database instance. See the TIBCO BusinessEvents Administration guide for details.

How to Run the Example

  1. Copy the database JDBC driver libraries to c:/tibco/be/5.0/lib/ext/tpcl.

  2. Import the TickerTracker project in TIBCO BusinessEvents® Studio.

  3. Update the SharedResources/JDBCConfig.sharedjdbc. Update the Database URL to point to your JDBC database. Also update the username and password to the login used for TIBCO BusinessEvents.

  4. Build the EAR file. Update the file location to BEStudioWorkspace/TickerTracker.ear.

  5. Open a command window and at the command prompt, generate the database creation scripts using the following command.

    cd BEStudioWorkspace
c:/tibco/be/5.0/bin/be-jdbcdeploy --propFile c:/tibco/be/5.0/bin/be-jdbcdeploy.tra -o TickerTracker BEStudioWorkspace/TickerTracker.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  6. Populate the database using the generated SQL scriptBEStudioWorkspace/TickerTracker.sql.

  7. Open a command window and at the command prompt, start the TIBCO BusinessEvents engine using the following command.

    cd c:/tibco/be/5.0/views/bin
be-views -u default -c BEStudioWorkspace/TickerTracker/TickerTracker/tickertrackerdb.cdd BEStudioWorkspace/TickerTracker.ear

    Adapt the command as needed, for example, if you have customized the example project, or are running it from a different directory.

  8. Launch the URL http://localhost:8080 using Internet Explorer.

  9. Login in to the dashboard using admin as Username and admin as Password.

  10. Create an ticker event and give it a price and volume. Repeat this step to generate ticker history, as desired.

    In the dashboard you should see the charts change to reflect the changes.

  11. To reset the data do the following:

    Run the generated SQL script c:/tibco/be/5.0/bin/tickertracker_cleanup.sql.

    Restart the engine at the command line.
